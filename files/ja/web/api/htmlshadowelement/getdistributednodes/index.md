**`HTMLShadowElement.getDistributedNodes()`** メソッドは、この `<shadow>` 要素に関連した {{glossary("distributed nodes")}} の静的な {{domxref("NodeList")}} を返します。

## 値

{{domxref("NodeList")}} です。

## 例

```js
// 分散ノードを取得
const nodes = myShadowObject.getDistributedNodes();
```
