**`keys()`** メソッドは [`values()`](/ja/docs/Web/JavaScript/Reference/Global_Objects/Set/values) メソッドのエイリアスです。

## 構文

```js-nolint
keys()
```

### 返値

与えられた `Set` の各要素の値を挿入順で含んだ新しい反復子オブジェクト。
