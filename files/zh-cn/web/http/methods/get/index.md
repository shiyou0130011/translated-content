{{HTTPSidebar}}

**HTTP `GET` 方法**请求指定的资源。使用 `GET` 的请求应该只用于获取数据。

| 请求是否有主体                   | 否  |
| -------------------------------- | --- |
| 成功的响应是否有主体             | 是  |
| {{Glossary("Safe/HTTP", "安全")}}     | 是  |
| {{Glossary("Idempotent", "幂等")}}     | 是  |
| {{Glossary("Cacheable", "可缓存")}} | 是  |
| HTML 表单是否支持                | 是  |

## 语法

```plain
GET /index.html
```
