`image` 是{{domxref("Notification")}} 接口的只读属性，包含了需要显示在通知信息里的图片的 URL，可通过{{domxref("Notification.Notification","Notification()")}}构造函数的 `image` 选项指定。

## 语法

```plain
var image = Notification.image;
```

### 值

{{domxref("USVString")}}。
