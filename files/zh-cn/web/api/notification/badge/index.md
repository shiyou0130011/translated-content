当没有足够的空间来显示通知本身时，{{domxref("Notification")}}接口的 **`badge`** 属性返回用于表示通知的图像 URL。

## 语法

```plain
var url = Notification.badge
```

### 值

包含一个 URL 的 {{domxref('USVString')}} 。
