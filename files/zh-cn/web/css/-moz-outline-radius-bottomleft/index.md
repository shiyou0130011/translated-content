{{Non-standard_header}}{{CSSRef}}

在 Mozilla 应用中， **`-moz-outline-radius-bottomleft`** [CSS](/zh-CN/docs/Web/CSS) 属性用于设置元素的左下角 {{cssxref("outline")}} 圆角。

{{cssinfo}}

## 示例

### HTML

```html
<p>看这段话的左下角。</p>
```

### CSS

```css
p {
  margin: 10px;
  border: solid cyan;
  outline: dotted green;
  -moz-outline-radius-bottomleft: 2em;
}
```
